Metro Guide App

Mobile App | Built with Flutter

Overview

The Metro Guide App is a cross-platform mobile application designed to simplify metro navigation for users in Cairo. Built entirely with Flutter, it focuses on intuitive design, real-time updates, and efficient state management.

Features

  • Station search and route navigation
  • Real-time updates for train schedules
  • Clean, user-friendly interface
  • State management with GetX

Tech Stack

Flutter, Dart, GetX, Firebase
